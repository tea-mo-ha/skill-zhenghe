# Project Codex Policy

- Prefer `skill-suite-orchestrator` as the entry skill for software tasks in this repository.
- Never include `skill-suite-orchestrator` itself in `chosen_subskills`.

## Page generation
- Allowed local skills only:
  - `frontend-design`
  - `frontend-ui-engineering`
  - `api-and-interface-design`
  - `vercel-react-best-practices`
  - `brainstorming` (only when requirements are ambiguous)
- Do not use `build-web-apps:*` skills in this repository.
- Do not use other external frontend plugin skills unless the user explicitly requests them.
